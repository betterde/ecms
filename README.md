## ECMS

一个面向代购的简单电商管理系统。前端采用 [Vue](https://vuejs.org) 和 [Element UI](https://element.eleme.io/) 实现，统计图表使用 [Antv G2](https://g2.antv.vision/en/) 实现，后端 API 采用 Laravel 实现。

- [x] 数据分析
- [x] 订单管理
- [x] 商品信息管理
- [x] 库存管理
- [x] 客户信息管理
- [x] 采购订单导出 Excel
- [] 代理商管理
- [] 支付系统
